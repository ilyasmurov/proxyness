## Design Context

### Users
- **Primary**: Developer who built it for himself, now expanding to family, colleagues, and paid subscribers
- **Audience range**: Tech-savvy developers to non-technical family members
- **Usage pattern**: Configure once (pick mode, toggle apps), then glance periodically to confirm connection. Occasional mode/app changes
- **Context**: Desktop utility (macOS + Windows), runs alongside other work. Users want confidence it's working without needing to think about it
- **Business model**: Subscription-based product, needs to feel worth paying for

### Brand Personality
- **Reliable** — rock-solid, always working, inspires confidence
- **Premium** — worth paying for, not a weekend hack
- **Technological** — serious engineering behind it, not dumbed down
- **Powerful** — full system-level proxy with per-app routing, not a simple toggle

### Aesthetic Direction
- **Reference**: Star Citizen launcher — sci-fi, serious, sleek, not gamified
- **Theme**: Dark + Light (both needed)
- **Palette direction**:
  - Dark gray-blue base (not pure black, not navy — somewhere industrial)
  - Warm amber/yellow accent like incandescent bulb glow (status, active states)
  - Cool blue for secondary accent (data, info, links)
  - Avoid: neon, cyberpunk, gaming aesthetics. Avoid: cheap VPN look with ads and clutter
- **Anti-references**: Cheap free VPN apps. Toy-like/playful interfaces. Generic "AI slop" dark dashboards with cyan gradients
- **Assets available**: Earth-from-space videos and images (rotation, golden glow) that can be used as background elements

### Design Principles
1. **Confidence at a glance** — Connection status must be readable in under 1 second. The primary use case is "open, confirm it's working, close"
2. **Progressive complexity** — Simple surface hides powerful controls. All Traffic mode shows nothing extra. Selected mode reveals the full app/site control panel. Browsers mode explains what's happening
3. **Earned premium** — Quality in details (typography, spacing, micro-interactions) not in decoration (gradients, glow, particles). A machined-metal feel, not a Christmas tree
4. **Serious tool, warm accents** — Cold structural palette warmed by amber indicators. Like cockpit instruments — functional but beautiful
5. **Respects attention** — No unnecessary animation. No repeated information. No empty space filled with decoration. Every element earns its place

### Functional Requirements
- **3 modes**: All Traffic (everything proxied, no config), Selected (toggle apps + sites on/off), Browsers Only (SOCKS5, no app routing)
- **Status info**: Connected/Disconnected, server IP, transport (UDP/SOCKS5), uptime, download/upload speed
- **App/site list**: Only visible in Selected mode. Each item has icon, name, domain, on/off toggle
- **Notifications**: Update available, maintenance, migration alerts
- **Settings**: Gear icon access
- **Window**: 840×580px, frameless, non-resizable
- **Platforms**: macOS + Windows

### Accessibility
- Standard font sizes (no special requirements)
- Standard contrast (WCAG AA sufficient)
- Both dark and light themes
