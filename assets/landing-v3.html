Proxyness

Прокси, которого не видно в сети

Весь трафик выглядит как обычный HTTPS. Один TLS-поток через порт 443 — для DPI и провайдера это обычный веб-сервер. Работает на macOS и Windows.

Возможности

Невидим на порту 443

Весь трафик идёт одним TLS-потоком через стандартный HTTPS-порт. Для любого DPI по пути это обычный веб-сервер с валидным сертификатом Let's Encrypt.

AutoTransport

Два транспорта с автоматическим выбором. Собственный UDP с ARQ и BBR/CUBIC для быстрого TTFB; TLS для тяжёлых загрузок. Без настроек.

Гибридный TUN + Браузер

В TUN-режиме приложения идут через системный тоннель, браузеры — через локальный SOCKS5. Или включи режим «только браузер», если этого достаточно.

Раздельный тоннель

Выбирай, какие приложения идут через прокси. Telegram и Discord — через тоннель, игры — напрямую. Всё настраивается в один клик.

Kill Switch

Три независимых детектора следят за тоннелем. При обрыве трафик блокируется мгновенно — реальный IP не утекает. Связь восстанавливается автоматически.

Привязка к устройству

Ключ привязывается к аппаратному отпечатку машины при первом подключении. Даже при утечке ключа — чужое устройство не пройдёт.

Автообновления

Клиент проверяет обновления автоматически и уведомляет о новых версиях. Обновление в один клик, без переустановки.

Нет логов

Сервер не хранит историю посещений, DNS-запросов или содержимого трафика. Только агрегированная статистика объёма для биллинга.

Что работает через прокси

Telegram, Discord, Claude, ChatGPT, YouTube, Instagram, Twitter / X, Notion, Figma, Spotify, Netflix, LinkedIn, Medium, GitHub Copilot, Steam (магазин), Midjourney, Slack, Zoom — и всё остальное, что не открывается без прокси. В TUN-режиме работает любое приложение. В браузерном — любой сайт.

Как подключиться

01

Установи приложение

Скачай установщик для macOS или Windows и запусти. Установка занимает меньше минуты. Не требуется ни командная строка, ни конфиги, ни расширения для браузера.

02

Введи ключ доступа

Вставь ключ, который ты получил при оформлении подписки. При первом подключении ключ автоматически привяжется к твоему устройству — никто другой не сможет им воспользоваться.

03

Нажми Connect

Готово. Трафик идёт через тоннель. Выбери режим TUN или «Только браузер» в шапке приложения — переключение в любой момент, без переподключения.

Технические характеристики

Протоколы TUN (системный тоннель), SOCKS5 (браузерный режим), UDP (кастомный транспорт с ARQ), TLS 1.3 (основной транспорт)
Порт 443 — стандартный HTTPS, один порт для всего трафика
Шифрование TLS 1.3 (AES-256-GCM / ChaCha20-Poly1305), HMAC-SHA256 аутентификация, Let's Encrypt сертификат
Congestion BBR и CUBIC с автоматическим выбором на UDP-транспорте
Платформы macOS (Apple Silicon), Windows (x64)
Клиент Electron + React, нативные установщики (.pkg / .exe), автообновления
Сервер Нидерланды, 4 vCPU, 8 GB RAM, 1 Gbps канал
Kill Switch Три независимых детектора (D1/D2/D3): обрыв транспорта, проверка сервера, детектор стагнации
Split tunnel Per-app маршрутизация через gVisor netstack: proxy_all_except или proxy_only
Привязка Аппаратный fingerprint (IOPlatformUUID / MachineGuid), бинарный протокол 0x03
