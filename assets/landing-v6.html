Характеристики

Технические детали, на которые можно положиться.

Proxyness / v1.31

По приглашению Бесплатно в бета-программе
  • Транспорт TLS 1.3 на порту 443 с мультиплексированием Единый порт обслуживает TCP, UDP и HTTP-админку. Отличить от обычного HTTPS на уровне провайдера невозможно.
  • Аутентификация HMAC-SHA256, привязка к устройству Клиентский ключ подписывает 41-байтное сообщение на каждое соединение. Машинный отпечаток фиксируется при первом подключении.
  • Архитектура TUN + gVisor netstack + SOCKS5 Приложения идут через TUN-устройство, браузеры — через локальный SOCKS5 на 127.0.0.1:1080. Гибридный режим работает одновременно.
  • Сервер Aeza NL · 4 CPU · 8 GB · 1 Gbps Нидерланды, Амстердам. Ядро Linux с BBR и fq_codel. Контейнер с лимитом 32 768 файловых дескрипторов.
  • Платформы macOS 12+ (Intel / Apple Silicon), Windows 10+ Нативный Electron-клиент с привилегированным хелпером для управления TUN-устройством. Авто-обновления через GitHub Releases.
  • Устойчивость Авто-восстановление соединения Три детектора обрыва: транспортный, проверка здоровья и детектор застоя. Переподключение без потери активных соединений.
  • Логирование Только метрики трафика, без контента Сервер считает байты и активные соединения на устройство. DNS-запросы, хосты и содержимое пакетов не записываются.
