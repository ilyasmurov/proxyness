§ Возможности
Стр. 002 / 006

Три режима.
Один клик.

Раздел 01

Маршрутизация Каждое приложение идёт по своему маршруту.

Выбирайте, какие приложения идут через прокси, а какие напрямую. TUN-режим на уровне системы перехватывает трафик каждого процесса отдельно — браузер видит один путь, мессенджер — другой.

Никаких системных расширений и правил файрвола. Настройка в один клик из интерфейса клиента.

Раздел 02

Браузерный SOCKS5 Локальный прокси, заточенный под веб.

Локальный SOCKS5-прокси для браузеров. Работает параллельно с TUN, обходит QUIC и обеспечивает максимальную совместимость с веб-сервисами.

Активируется автоматически при подключении — без ручной настройки PAC-файлов и системных переменных.

Раздел 03

Раздельное туннелирование Вы решаете, что идёт через туннель.

Два режима: проксировать всё, кроме выбранных приложений, или проксировать только выбранные. Гибкая настройка для каждого процесса в системе.

Идеально для геймеров, разработчиков и всех, кто хочет снизить задержку на локальных сервисах.
