Маршрутизация по приложениям

Выбирайте, какие приложения идут через прокси, а какие напрямую. TUN-режим на уровне системы перехватывает трафик каждого процесса отдельно.

Браузерный SOCKS5

Локальный SOCKS5-прокси для браузеров. Работает параллельно с TUN, обходит QUIC и обеспечивает максимальную совместимость с веб-сервисами.

Раздельное туннелирование

Два режима: проксировать все, кроме выбранных, или проксировать только выбранные. Гибкая настройка для каждого приложения.
