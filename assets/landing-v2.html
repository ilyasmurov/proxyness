PROXYNESS

Невидим для DPI. Неотличим от HTTPS.

Системный TUN-тоннель и браузерный SOCKS5 через один TLS-поток на порту 443. Для провайдера — обычный веб-сервер с сертификатом Let's Encrypt.

Порт 443

Весь трафик идёт одним TLS-потоком через стандартный HTTPS-порт. Для DPI по пути — обычный веб-сервер.

Гибридный TUN + SOCKS5

Приложения через системный тоннель, браузеры через SOCKS5. Или только один из режимов — на ваш выбор.

Раздельный тоннель

Telegram и Discord через прокси, игры напрямую. Per-app маршрутизация через gVisor netstack.

Kill Switch с тремя детекторами

D1 — обрыв транспорта, D2 — периодическая проверка сервера, D3 — детектор стагнации. Мгновенная блокировка при обрыве, автоматическое восстановление. Реальный IP не утечёт.

Привязка к устройству

Аппаратный fingerprint при первом подключении. Даже при утечке ключа чужое устройство не пройдёт.

Автообновления

Клиент проверяет новые версии и уведомляет. Обновление в один клик, без переустановки.

Нет логов

Сервер не хранит историю посещений, DNS-запросов или содержимого. Только агрегированный объём для биллинга.
