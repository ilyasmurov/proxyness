# UDP No-ARQ Transport Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove ARQ (reliability layer) from the UDP transport data path. Data packets are encrypted and sent fire-and-forget. Stream open uses simple retry. Delete ~3400 lines of ARQ code.

**Architecture:** Strip ARQ from both daemon (transport/udp.go) and server (udp/listener.go). Simplify packet format by removing PktNum. Data flows directly: encrypt → send → forget. Stream open uses retry (send, wait 1s for result, retry up to 3 times). gVisor TCP handles retransmission for TCP proxy traffic; UDP proxy traffic tolerates loss natively.

**Tech Stack:** Go 1.26, XChaCha20-Poly1305 (existing), X25519 (existing).

**Important:** Tasks 1–3 remove all ARQ usage. Task 4 deletes the ARQ package and is the first point where `go build` succeeds. Do NOT attempt to build between tasks 1–3.

- [ ] **Step 6: Rewrite OpenStream() — direct send with retry for TCP**

Replace the ARQ send logic in OpenStream(). The new flow:

For TCP streams (`streamType == 0x01`):
1. Send MsgStreamOpen directly
2. Wait for first MsgStreamData on recvCh (this is the result byte: 0x01 ok / 0x00 fail)
3. If no response in 1s, retry (max 3 attempts)

For UDP streams (`streamType == 0x02`):
1. Send MsgStreamOpen directly
2. Return immediately (no need to wait for connect ACK)

Replace the send+wait section with:

```go
// Encode stream open payload: streamType + addrType + addr + port
payload := pkgudp.EncodeStreamOpen(streamType, addr, port)

if streamType == 0x01 { // TCP — need to know if dial succeeded
	var result byte
	var gotResult bool
	for attempt := 0; attempt < 3; attempt++ {
		t.sendPacketDirect(pkgudp.MsgStreamOpen, id, payload)
		select {
		case data := <-s.recvCh:
			if len(data) > 0 {
				result = data[0]
				gotResult = true
			}
		case <-time.After(time.Second):
			continue
		case <-t.done:
			return nil, ErrTransportClosed
		}
		if gotResult {
			break
		}
	}
	if !gotResult {
		t.mu.Lock()
		delete(t.streams, id)
		t.mu.Unlock()
		return nil, fmt.Errorf("stream open timeout after 3 attempts")
	}
	if result != 0x01 {
		t.mu.Lock()
		delete(t.streams, id)
		t.mu.Unlock()
		return nil, fmt.Errorf("stream open rejected")
	}
} else { // UDP — fire and forget
	t.sendPacketDirect(pkgudp.MsgStreamOpen, id, payload)
}
```

Remove all `t.arq.CreateRecvBuffer()` and `t.arq.Send()` calls.

```
- **UDP transport is fire-and-forget (no ARQ)**: As of server-2.2.0, the UDP transport sends encrypted packets without a reliability layer. Lost data packets are handled by upper layers: gVisor TCP retransmits for TCP proxy traffic; UDP proxy traffic (games, voice) tolerates loss natively. Only stream open/close uses simple retry (3 attempts × 1s). The previous ARQ implementation (BBR congestion control, per-stream reordering, retransmit/ACK loops) was removed because it caused retransmit storms under packet loss and added latency for game traffic. Packet format: QUIC-disguised outer frame + XChaCha20-Poly1305 encrypted inner payload (Type + StreamID + DataLen + Data, 7 bytes overhead).
```
